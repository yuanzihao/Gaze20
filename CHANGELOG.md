# Changelog

## [0.3.0] - 2026-06-20

### Added
- 新增每日眼睛症状自评弹窗，支持眼睛红血丝、干涩、视物模糊、头痛不适、颈肩酸痛 5 项评分。
- 新增工程模式入口，可在设置页手动预览提醒弹窗、症状自评并调整调试分值。
- 新增 GitHub Actions 发布工作流，推送 `vX.Y.Z` tag 后可自动生成安装包和免安装 zip。

### Changed
- 症状自评纳入风险模型与每日聚合，`SCHEMA_VERSION` 升级到 7。
- 多屏休息提醒改为同步的独立 WebView 卡片，倒计时、按钮状态和动画保持一致。
- README 与产品化文档更新为当前实现，补齐隐私、本地存储、工程模式、CI 和发布说明。

### Fixed
- 修复休息提醒与症状自评同时触发时的冲突，自评会在休息提醒结束后顺延。
- 修复多屏提醒空白白框、窗口闪烁和不同分辨率下卡片比例变形的问题。
- 修复设置页在小窗口下活动检测控件、导出导入区偏位变形的问题。
